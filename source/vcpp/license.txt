Quick installation notes for the "./buildEnv" folder:

	* Install GitHub Desktop/a command line Git client.
	* Install a command-line SVN client.
	* Install 7-Zip.
	* Include git.exe, svn.exe, 7z.exe in your system path variable.
	* Install NSIS, minimum version 3.0.
